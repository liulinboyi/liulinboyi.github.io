# Node.js以及Vue-cli安装

# 1. Node.js 下载 与 安装

Node.js 官网 地址：官网

# 2. 检测Node.js是否安装成功

win + r 输入cmd 进入 cmd 控制台 输入 node -v 如果有版本输出 就说明成功了！

# 3. Vue-cli 安装

官方网址官网

安装

npm install -g @vue/cli
# OR
yarn global add @vue/cli

检测是否安装成功

vue -V

创建一个项目

vue create my-project
# OR
vue ui

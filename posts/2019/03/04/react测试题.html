# React测试题

假设页面中已经有一个 id 为 root 的 div，我希望把 <h1>Hello, world!</h1> 展示在页面里，代码如下

________A________(
  <h1>Hello, world!</h1>,
  document.getElementById('root')
);

答案：ReactDOM.render

function ComponentA(){
    return
      <div>hi</div>
}

请问 ComponentA() 的返回值是

  1. undefined
  2. 一个 React 元素，对应一个 div

答案： 1

解析return ()//一定要加小括号或者紧跟其后

function ComponentA(props){
    return(
        <div style=_______B________>hi</div>
    )
}

function App(){
    return (
        <div>
            <ComponentA width={100}/>
        </div>
    )
}

需求是让 ComponentA 的 div 的宽度与 props.width 一致，请问 _____B______ 处应该怎么填？ 代码运行处：https://codesandbox.io/s/01z20q1p8w

答案：{{width:props.width}}

function ComponentA(props){
    const {children, ...rest} = props
    return(
        <div>
            <div> A 组件</div>
            <ComponentB ______C______/>
        </div>
    )
}

function ComponentB(props){
    console.log(props)  // 注意这里
    return(
        <div>
            <div> B 组件</div>
        </div>
    )
}

function App(){
    return (
        <div>
            <ComponentA width={100} height={100} other={404}>
              A
            </ComponentA>
        </div>
    )
}

需求是让 ComponentA 把自己接收到的所有属性都传递给 ComponentB，请问 _____C______ 处应该怎么填？ 也就是说代码中的 console.log(props) 必须打印出 {width: 100, height: 100, other: 404} 代码运行处：https://codesandbox.io/s/z6q431v44l

  1. props
  2. rest
  3. ...props
  4. ...rest
  5. {...props}
  6. {...rest}

答案：6

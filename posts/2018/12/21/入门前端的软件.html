# 入门前端的软件

# 1.如何安装nodejs

首先访问nodejs的官网nodejs官网

# 2.下载 nodejs 的安装包

下载nodejs安装之后会同时把npm安装上

在国内npm装包会比较慢

可以使用国内的淘宝镜像https://npm.taobao.org/

# 设置 ：

# 命令行安装npm 和 cnpm 以及 nrm
npm install -g cnpm --registry=https://registry.npm.taobao.org

或者安装nrm

cnpm i -g nrm

# 如果想体验比较好的安装包的体验，可以使用 yarn

yarn
